this function estimates the value of kappa for a gamma distribution, using the secant method.

Parameters
xCurrent value for kappa (input/output)
diffCurrent value for the discrepancy between the CDF evaluated using the current kappa parameter value and the 95th percentile. (output)
paramsPointer to an array of parameters.
