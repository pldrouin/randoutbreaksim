225  printf("Stochastic simulation of outbreaks, using gamma distributions for the different time periods and a Poisson distribution for the number of interaction events where transmission can occur.\n");
226  printf("\nBasic reproduction parameters:\n\n");
227  printf("\tThe basic reproduction number R0 is defined by the expression\n");
228  printf("\t\tR0 = lambda * tbar * (g_ave - 1) * pinf.\n");
229  printf("\n\tA sufficient number of input parameters must be provided to determine, without overdetermining, the above expression.\n");
230  printf("\tThe mu and p parameters are alternate parameters that can be provided instead of g_ave.\n");
231  printf("\tmu is the mean of an unbounded logarithmic distribution with parameter p (mu = -p / ((1 - p) * log(1 - p))).\n");
232  printf("\tThe expression of g_ave as a function of p depends on the type of group distribution that is selected for the events.\n");
233  printf("\tAn event is defined to include at least two invitees.\n");
234  printf("\n\t--group_log_attendees_plus_1, the default distribution from branchsim, indicates that the number of attendees in an event is to be distributed according to a logarithmically-distributed variable plus 1. For an infinite population, a fixed communicable period, and when pinf=1, this results in the total number of infections from a given infectious individual to follow a negative binomial distribution. The expression for g_ave with this distribution is\n");
235  printf("\t\tg_ave = mu + 1.\n");
236 
237  printf("\n\t--group_log_attendees indicates that the number of attendees in an event is to be distributed according to a logarithmically-distributed variable (truncated below 2). The expression for g_ave with this distribution is\n");
238  printf("\t\tg_ave = -p * p / ((1 - p) * (log(1 - p) + p)).\n");
