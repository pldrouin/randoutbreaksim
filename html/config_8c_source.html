339  printf("\tAn event is defined to include at least two invitees.\n");
340  printf("\n\t--group_log_plus_1, the default distribution from branchsim, indicates that the number of invitees/attendees in an event is to be distributed according to a logarithmically-distributed variable plus 1. The expression for g_ave with this distribution is\n");
341  printf("\t\tg_ave = mu + 1.\n");

343  printf("\n\t--group_log indicates that the number of invitees/attendees in an event is to be distributed according to a logarithmically-distributed variable (truncated below 2). When using group_attendees, the expression for g_ave with this distribution is\n");
344  printf("\t\tg_ave = -p * p / ((1 - p) * (log(1 - p) + p)).\n");
