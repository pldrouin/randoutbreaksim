Finds the root of a function depending on a single variable.

This function searches for the root of the user-provided function.

Parameters
rfhandle for the root finding algorithm.
epsStop point for the algorithm. The algorithm stops once the absolute value for the second argument from the user-defined function is smaller than eps.
maxiterMaximum number of iterations for the algorithm.
xminMinimum value for the location of the root.
xmaxMaximum value for the location of the root.
xInitial estimate for the root location, and location of the computed root.
Returns
0 if successful and a non-zero value otherwise.
