Finite logarithmic deviate.

Returns a logarithmic deviate. Returned value is finite, and can be cast to the desired integer data type. Modified algorithm from "Non-Uniform Random Variate Generation ", by Luc Devroye.

Finite logarithmic deviate with a lower bound of 2.

Returns a logarithmic deviate greater than 1. Returned value is finite, and can be cast to the desired integer data type. Modified algorithm from "Non-Uniform Random Variate Generation" by Luc Devroye. Loop seems necessary. Note: There is no support for p=0 by this function due to optimisation.
