Generates one function that generates a specific combination of communicable and latent periods.

This preprocessing macro generates a function that generates the communicable period for the current infectious individual and a latent period, as determined by the simulation variables and parameters. It considers the alternate communicable period based on the q simulation parameter, as well as the two types of interrupted periods. It also sets the infectious_at_tmax parameter for the current infectious individual.

Parameters
LATENTType of latent period (0=none, 1=fixed, 2=variable)
MAINType of main period (1=fixed, 2=variable)
ITType of interrupted main period (0=none, 1=fixed, 2=variable)
ALTERNATEType of alternate period (0=none, 1=fixed, 2=variable)
IMType of interrupted alternate period (0=none, 1=fixed, 2=variable) latent)
